\# BH-2026 AI图像真假检测 小组代码仓库

\## 仓库文件说明

1\. image\_detection.ipynb

完整阶段1数据处理代码，包含抽样核验、可视化校验，用于作业验收、数据bug排查

2\. train\_pipeline.py

精简交付脚本，仅保留训练必需的数据加载逻辑，队友写模型直接导入使用

3\. environment.yml

统一Python依赖配置，本地一键复现运行环境

4\. .gitignore

过滤缓存、图片、数据集、权重等大文件，不提交到仓库



\## 数据集重要提醒

本仓库**仅存放代码，不含图片数据集**

1\. Kaggle使用限制：队友无法加入竞赛，无法自动挂载数据集

2\. 本地运行操作：

&#x20;  - 去竞赛页面下载完整数据集压缩包

&#x20;  - 解压后目录层级必须和云端完全一致，仅修改train\_pipeline.py里本地DATA\_ROOT路径



\## 数据规则

1\. 输入张量尺寸：(B, 3, 32, 32)

2\. 标签定义：0 = REAL真实图片，1 = FAKE AI生成图

3\. 数据集划分：全部训练集9:1拆分，固定seed=42保证实验可复现

4\. 数据增强：训练集带随机裁剪+水平翻转；验证/测试集无随机增强



\## 协作规范

1\. 每次修改代码前先执行 git pull 同步最新版本

2\. 提交备注规范：feat(新增功能) / fix(修复bug)

3\. 分工隔离：data相关文件仅数据负责人修改，模型代码新建独立文件不改动数据脚本
